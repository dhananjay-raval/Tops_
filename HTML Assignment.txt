Q.1  Are the HTML tags and elements the same thing?

Ans. HTML tags and elements are related but not exactly the same thing. HTML tags are the markup symbols used to define elements within an HTML document. Elements are comprised of a start tag, content, and an end tag, whereas tags are individual components that define the structure of elements. So, while tags are used to create elements, elements include both the opening and closing tags along with the content.

Q.2 What are tags and attributes in HTML?
Ans. In HTML, tags are the markup symbols used to define elements within a document. They consist of angle brackets (< and >) enclosing a keyword, like <p> for a paragraph. Attributes, on the other hand, provide additional information about an element and are placed within the start tag. They are comprised of a name and a value and are used to modify the behavior or appearance of an element, such as <img src="image.jpg" alt="description">, where "src" and "alt" are attributes

Q.3  What are void elements in HTML? With Example.

Ans. Void elements in HTML are elements that do not have closing tags and do not have any content. They are self-closing tags. Examples of void elements include <img>, <br>, <input>, and <meta>.

Q.4.  What are HTML Entities? With Example.
Ans.HTML entities are special codes used to represent characters that have special meaning in HTML, such as reserved characters like <, >, &, or characters with accents or diacritics. They are often used when you want to display these characters in your HTML code without them being interpreted as markup.

For example:

&lt; represents the less than symbol (<).
&gt; represents the greater than symbol (>).
&amp; represents the ampersand symbol (&).
&eacute; represents the letter "é".

Q.7What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML
elements? With Example.

Ans.The 'id' attribute is unique and used to identify a single element, while the 'class' attribute can be applied to multiple elements to group them together for styling or behavior.

Q.8.What are the various formatting tags in HTML?
Ans.HTML provides various formatting tags such as <h1> to <h6> for headings, <p> for paragraphs, <b> or <strong> for bold text, <i> or <em> for italic text, <u> for underline (not recommended), <s> or <strike> for strikethrough, <sup> for superscript, <sub> for subscript, <pre> for preformatted text, <code> for code, <blockquote> for block quotes, <address> for contact information, and <hr> for horizontal rules.

Q.10 What is the difference between a block-level element and an inline element?

Ans.Block-level elements start on a new line and take up the full width available, while inline elements do not start on a new line and only take up as much width as necessary.

Q.11How to create a Hyperlink in HTML? With Example.
Ans.To create a hyperlink in HTML, use the anchor tag <a> with the href attribute to specify the URL. For example: <a href="https://www.example.com">Link Text</a>.
